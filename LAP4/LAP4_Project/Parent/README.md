# Welcome to PinPlace

To run the repo code:

1. Fork and clone this repo
2. Open the client:

   - `cd [client]`
   - `npm install`
   - `npm run build`

3. Start a server and database combo
   - `cd [Server]`
   - `docker-compose up`
   - to stop and keep data `docker-compose down`
   - to stop and remove all artifacts `docker-compose down --volumes --remove-orphans`
